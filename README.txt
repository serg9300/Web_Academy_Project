WebAcademyChatBot is a simple non-commercial chat bot, created as a final project for Web Acedmy (Python Course)

All necessary libraries and packages you may find in the file 'requirements.txt'
Project were deployed on heroku server

Functionality:
1. Bot has intigrated AI small-talk. All chat should be typing in english.
2. Bot has photo requested command with a funny dogs. You should type '/doggy' for testing
